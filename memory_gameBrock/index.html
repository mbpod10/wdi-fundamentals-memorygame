Instructions

Memory Game is a game in which all of the cards are laid face down on a surface and two cards are flipped face up over each turn. The object of the game is to turn over pairs of matching cards. Remember to use your memory!

Why Memory Game?

Excercising the brain is one of the most important things to do. Not only is this game fun but it is healthy for you! Check out what the developer has to say about it here.
